Site Description

The purpose of this app is to provide a convenient API for storing, sharing, and learning different recipes. You can use this app for any recipes that requires directions and ingredients, from coffees to quiches. You can do this by posting a recipe, and then adding ingredients linked to the recipe. You can choose to set your recipe to public or private. Only you can see private recipes, but anyone can view public recipes. You can user-info this to learn new recipes! For full usage instructions, please read the documentation linked above.

JWT Instructions

To use this site, first ensure you have a working Google account. Then, click the "JWT Generation" link at the top of this page, and follow the steps to login. Once authenticated, you will be redirected to a page that displays your JWT. Copy it, and include it in the "Authorization" header as "Bearer <Your JWT>".
You are now ready to start making requests to the api. Please see the "Usage Documenation" link at the top of this page for full documenation on all routes in the API.
